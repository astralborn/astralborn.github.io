🌉 WSL Port Bridge

Working in WSL2 means ports don't automatically mirror to Windows — every network change wipes your forwarding rules. This utility watches for network changes and re-applies port rules automatically, so you can just work.

# Re-applies rules on every network change
def sync_ports(rules):
  for rule in rules:
    forward_port(rule.host, get_wsl_ip(), rule.wsl)
